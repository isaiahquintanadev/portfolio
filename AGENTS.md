# AGENTS.md

## Idioma de trabajo

Responde siempre en español.

## Proyecto

Este proyecto es mi portfolio personal desarrollado con Next.js.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel para despliegue

## Objetivo general

Quiero mejorar el portfolio manteniendo una estética profesional, limpia, moderna y cuidada.

La web debe transmitir perfil de desarrollador full-stack, con especial atención a:

- claridad;
- buen diseño responsive;
- buen rendimiento;
- código limpio;
- componentes reutilizables;
- buenas prácticas de Next.js.

## Preferencias de desarrollo

- No hagas cambios innecesarios.
- No reestructures todo el proyecto si no es imprescindible.
- Antes de cambiar arquitectura, explica el motivo.
- Mantén el código simple y fácil de mantener.
- Usa TypeScript correctamente.
- Evita soluciones excesivamente complejas.
- Respeta el estilo visual actual salvo que se pida lo contrario.
- Prioriza responsive mobile-first.
- Comprueba que no se rompa el build.

## Diseño

El diseño debe ser:

- moderno;
- elegante;
- profesional;
- responsive;
- legible;
- con buena jerarquía visual;
- sin sobrecargar de animaciones.

## Comandos importantes

Antes de dar por terminado un cambio, intenta ejecutar:

```bash
npm run lint
npm run build
```
